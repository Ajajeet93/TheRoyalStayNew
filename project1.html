The Royal Lotus – Hotel Management System

Overview

An end-to-end hotel management system that serves guests and administrators. This comprehensive web application allows guests to browse rooms, book stays, and submit inquiries, while providing a powerful administrative dashboard for managing all aspects of the hotel's operations, including bookings, room inventory, and guest messages.

The inviting landing page of The Royal Lotus.

Technologies Used

  • Backend: PHP (PDO) for secure and efficient database interactions.
  • Database: MySQL/MariaDB for structured data storage.
  • Frontend: Tailwind CSS for a utility-first and responsive design.
  • Icons: Font Awesome for a modern and clean user interface.

Key Features

For Guests

  • Browse and book rooms with secure validation.
  • View personal booking history and manage profile.
  • Explore the restaurant menu and submit contact messages.

Guest view of the room booking interface.

For Administrators

  • Manage all guest bookings (confirm, cancel, view details).
  • Full CRUD (Create, Read, Update, Delete) functionality for rooms and room types.
  • Handle and respond to all guest contact messages.

The comprehensive Admin Dashboard for managing bookings and rooms.

Authentication

The system includes a secure authentication module where users can register with unique credentials and log in to access booking features. Admins log in separately with elevated privileges to manage hotel operations.

  • Registration validates unique email and strong password.
  • Passwords stored securely using password_hash().
  • Login establishes a secure session with role-based access (User/Admin).

Security and Validation

The project prioritizes security by using prepared statements via PDO to prevent SQL injection, password_hash() for secure password storage, and role-based access control to protect sensitive admin pages.
